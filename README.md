# Smart Greenhouse Monitoring System

## Опис

Програма моделює автоматизовану систему контролю мікроклімату теплиці.

Функціонал:

- Аналіз температури
- Контроль вологості
- Аварійний режим
- Стабілізація середовища

## Виявлені проблеми коду

- Використання магічних чисел (16, 28, 45, 35, 75)
- Підвищена цикломатична складність у processEnvironment
- Клас виконує кілька відповідальностей
- Логіка прийняття рішень може бути винесена в окремі сервіси
